    Authentication

    This page provides blogs for setting up different authentication from your SAP system to Google Cloud. This is pre-requisite for ABAP SDK to work.

    There are various authentication options that depend on how your SAP system is hosted and which APIs you would want to connect. For official documentation refer the Authentication Guide
